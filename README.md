# 🌟 Eluxo – E-Commerce React App

**Eluxo** is a modern, premium e-commerce platform built with **React** and **Vite**. It delivers a seamless shopping experience for luxury, fashion, tech, and lifestyle products. The app features user authentication, dynamic product browsing, a clean checkout flow, and a stylis UI.

---

## ✨ Features

- 🛍️ **Product Catalog**  
  Browse a wide range of products with filtering by category, audience, and price.

- 🔍 **Product Details**  
  View detailed information, images, and features of each product.

- 🔐 **Authentication**  
  Sign-up and login system with local storage session handling.

- 🛒 **Shopping Cart**  
  Add, remove, and update product quantities in your cart.

- 💳 **Checkout**  
  A simple and intuitive checkout flow.

- 📰 **Blog Page**  
  Stay updated with the latest trends, product highlights, and company news.

- 📄 **About Page**  
  Learn more about Eluxo’s mission, values, and unique selling points.

- 📱 **Responsive Design**  
  Beautiful layout on desktop, tablet, and mobile screens.

- ⭐ **Best Sellers**  
  A highlighted section for top-selling or trending products.

- 📬 **Newsletter Subscription**  
  Collect user emails to keep them informed about new deals.

---

## 🧰 Tech Stack

| Layer         | Tools / Libraries                              |
|---------------|-------------------------------------------------|
| **Frontend**  | React 19, Vite, React Router DOM                |
| **UI**        | Material UI, Emotion, custom CSS                |
| **State**     | React Hooks, local component state              |
| **Backend**   | JSON Server (for mocking API via `db.json`)     |
| **Linting**   | ESLint                                          |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/eluxo.git

# Navigate into the project
cd eluxo

# Install dependencies
npm install

# Start the JSON server (mock API)
json-server --watch db.json --port 3000

# Start the Vite development server
npm run dev
